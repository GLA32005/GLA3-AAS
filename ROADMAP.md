# AgentSec 未来演进路线图 (Roadmap) 🛡️🚀

AgentSec 目前已完成 **v2.0 (Production Ready)** 阶段的交付，实现了全链路的数据持久化、RBAC 基础框架及真实的静态扫描能力。
以下是迈向 **v3.0 (Enterprise/Financial Grade)** 的未竟事宜与进阶建议：

## 1. 账户运维中心 (Identity & User Ops) 👤
- [ ] **用户增删改查 UI**：目前仅支持通过数据库 Seeding 或 `login` 逻辑自动创建 admin，需补全完整的用户管理界面。
- [ ] **SSO/SAML 集成**：对接企业级身份认证系统（如 LDAP、OIDC、钉钉/飞书扫码）。
- [ ] **密码策略强化**：实现强密码校验、MFA（多因素认证）及登录失败锁定逻辑。

## 2. 实时治理体验 (Real-time Governance) ⚡
- [ ] **WebSocket 全双工推流**：将目前的 15s 轮询升级为 WebSocket 实时推送，确保风险发生后毫秒级在控制台弹窗。
- [ ] **离线 SDK 热加载**：目前配置变更需 SDK 下次同步生效，未来可实现控制台点对点「指令下发」实现准瞬时拦截。

## 3. 深度安全引擎 (Advanced Security Core) 🧠
- [ ] **AST 抽象语法树扫描**：由目前的 Regex 扫描升级为深度 AST 分析，识别更复杂的逻辑注入与数据泄露路径。
- [ ] **LLM 研判助手**：集成私有 LLM 模型，对每一条告警进行自动化的语义研判，并在详情页生成「攻击意图总结」。
- [ ] **漏洞自动化修复**：针对扫描出的过度权限配置，提供「一键生成补丁」功能，自动裁剪 Agent 的工具权限。

## 4. 大规模集群支持 (Scalability & High Availability) d🌐
- [ ] **多租户/组织隔离**：完善 Team/Org 维度的多租户数据强隔离逻辑。
- [ ] **日志自动回滚与存档**：针对海量审计日志，实装基于时间的自动清理与冷存储存档逻辑（WORM 归档）。
- [ ] **异地灾备部署**：支持控制台的跨机房主从备份，确保治理系统本身的高可用。

## 5. 交互与体验 (UX & Polish) 🎨
- [ ] **全局搜索 (Command + K)**：实装跨模块的全量检索能力。
- [ ] **工单流转系统**：将告警处置升级为「流程驱动」，支持安全负责人派发任务给开发负责人进行修复跟进。

---
> AgentSec: 让每一个 AI Agent 都能在安全护栏内释放潜能。
